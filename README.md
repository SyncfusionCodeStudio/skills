# Skills

A public repository of agent skills for workflow automation and integrations. These skills are intended to be useful to **all AI agent users** and can be shared, reused, and extended across projects.

> Learn more about authoring and using skills in Code Studio:
> [https://help.syncfusion.com/code-studio/customization/skills](https://help.syncfusion.com/code-studio/customization/skills)

---

## Available Skills

| Skill | Description | Link |
|-------|-------------|------|
| Azure DevOps Ops | Run Azure DevOps work-item operations (list, get, create, update, delete) from chat using a PowerShell script and a secure PAT flow. | [skills/azure-devops-ops](skills/azure-devops-ops) |
